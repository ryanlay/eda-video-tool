About This Platform

What is EDA?

Electrodermal Activity (EDA), also known as Galvanic Skin Response (GSR), measures the electrical conductance of the skin, which varies with moisture level. EDA is widely used to measure:

  • Stress levels - Higher EDA indicates increased stress or arousal
  • Emotional responses - Peaks correspond to emotional events
  • Cognitive load - Mental effort affects skin conductance
  • Engagement - Attention and focus patterns

CSV File Format

Your CSV file should contain at minimum:

  • A timestamp column (e.g., timestamp_iso, timestamp, time)
  • An EDA data column (e.g., eda_scl_usiemens, eda)

Technology Stack

This platform is built with:

  • Flask - Python web framework (translates to ASP.NET MVC)
  • Plotly - Interactive visualizations
  • Bootstrap 5 - Responsive UI
  • Pandas - Data processing
Migration Ready: This application is structured to easily translate to ASP.NET Core MVC with minimal code changes. Routes map to Controllers, templates to Razor Views, and data processing to Services.
